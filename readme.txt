Generate AI images directly from the WordPress editor and add them to your media library.

== Description ==

Image Block for Pollinations.ai allows you to generate images using AI technology directly within the WordPress block editor. Simply enter a text prompt describing the image you want to create, and the plugin will generate it for you.

**Features:**

* Generate AI images from text prompts
* Save generated images to your WordPress media library
* Insert AI-generated images directly into your content
* User-friendly interface integrated into the block editor
* Automatically sets image alt text based on your prompt

This plugin uses the Pollinations.ai API to generate images based on your text descriptions.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/image-block-for-pollinations-ai` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Image Block for Pollinations.ai in the block editor to start generating images

== Frequently Asked Questions ==

= How do I generate an image? =

1. Add the 'Image Block for Pollinations.ai' to your post or page
2. Enter a descriptive text prompt in the text area
3. Click "Generate Image"
4. When the image appears, click "Save and insert" to add it to your content and media library

= Are there any limits to image generation? =

The prompt text is limited to 475 characters. Image generation speed depends on the Pollinations.ai API response time.

= Is this plugin free to use? =

Yes, this plugin is completely free to use as it relies on the free tier of the Pollinations.ai API.

This plugin uses the AI image generation. When you enter a prompt, that text is sent to Pollinations.ai. The generated images are saved to your WordPress media library.
